# SOC & Blue Team Labs

Repositorio de laboratorios prácticos orientados a SOC Analyst Level 1 y Blue Team.

Este espacio reúne investigaciones técnicas realizadas en distintas plataformas de entrenamiento, documentando metodología, evidencias, comandos utilizados, hallazgos, indicadores de compromiso y conclusiones defensivas.

## Plataformas incluidas

- BlueTeamLabs
- CyberDefenders
- TryHackMe
- LetsDefend

## Objetivo

El objetivo de este repositorio es construir un portfolio técnico enfocado en ciberseguridad defensiva, demostrando capacidades relacionadas con:

- Análisis de malware
- Análisis de tráfico de red
- Investigación de incidentes
- Identificación de indicadores de compromiso
- Detección de amenazas
- Uso de herramientas de línea de comandos
- Documentación técnica orientada a SOC
